# Spotify Playlist Organizer

A full-stack web application that helps users organize their Spotify library by creating playlists based on vibes and moods.

## Features
- OAuth 2.0 authentication with Spotify
- Filter songs by vibe (Happy, Sad, Energetic, Chill, Party)
- Create playlists directly in Spotify
- View all saved tracks with album artwork

## Tech Stack
- **Frontend:** React, Axios
- **Backend:** Node.js, Express
- **APIs:** Spotify Web API
- **Deployment:** Vercel (frontend), Render (backend)
